# 代码生成功能

AI 驱动的代码生成工具，支持根据模块描述自动生成前后端代码。

## 功能流程

### 1. 创建模块
- 输入模块名称（如"用户管理"）
- 输入模块标识（如"user"）
- 详细描述模块功能（AI 将根据此描述生成表结构）

### 2. 设计表结构
- **AI 生成**：点击"AI 生成表结构"按钮，根据模块描述自动生成数据库表结构
- **手动编辑**：可以添加、编辑、删除字段
- **字段配置**：
  - 基本信息：字段名、类型、长度、注释
  - 约束：可为空、主键、唯一、索引
  - 默认值

### 3. 生成代码
- **后端代码**：
  - Entity（实体类）
  - DTO（数据传输对象）
  - Controller（控制器）
  - Service（服务层）
- **前端代码**：
  - List（列表页面）
  - Form（表单组件）
  - Types（TypeScript 类型定义）
  - API（接口调用）

### 4. 下载使用
- 在线预览生成的代码
- 复制代码到剪贴板
- 下载完整代码包

## 文件结构

```
codeGen/
├── index.tsx           # 入口文件
├── types.ts            # 类型定义
├── ModuleList.tsx      # 模块列表主页面
├── ModuleForm.tsx      # 创建/编辑模块表单
├── TableDesigner.tsx   # 表结构设计器
├── CodePreview.tsx     # 代码生成预览
└── README.md           # 说明文档
```

## 使用说明

1. **访问页面**：导航到"代码管理 > 代码生成"
2. **创建模块**：点击"创建模块"按钮，填写模块信息
3. **设计表结构**：点击"设计表"按钮，使用 AI 生成或手动配置表结构
4. **生成代码**：确认表结构后，点击"生成代码"按钮
5. **下载代码**：预览并下载生成的代码

## 路由配置

需要在菜单管理中添加以下路由：

| 路径 | 组件 | 说明 |
|------|------|------|
| /code-manage/code-gen | ModuleList | 模块列表（主页面） |
| /code-manage/code-gen/design/:moduleId | TableDesigner | 表结构设计 |
| /code-manage/code-gen/preview/:moduleId | CodePreview | 代码预览 |
